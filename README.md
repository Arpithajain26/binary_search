# Binary Search – Complete Practice Guide

## 📌 Overview
This repository contains a **structured, day-wise collection of Binary Search problems**, implemented in Python.

Each problem is organized into its **own folder** with:
- Clean code
- A dedicated README explaining intuition
- Step-by-step logic

The goal of this repository is to **master Binary Search patterns** through consistent daily practice and clean Git workflow.

---

## 🧠 Why Binary Search?
Binary Search is one of the most important algorithms in **Data Structures & Algorithms**.

It is used when:
- The input data is **sorted**
- You want to reduce time complexity from **O(n)** to **O(log n)**

Binary Search is heavily tested in:
- Coding interviews
- Competitive programming
- System design foundations
